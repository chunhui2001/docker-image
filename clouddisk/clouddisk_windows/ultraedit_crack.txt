1. 安装完成后把UltraEdit软件关闭
2. 把网络给断掉，如果是笔记本WLAN也需要关闭。在网络连接里，把有线和无线网卡选中右击禁用，待图标都变成灰色后，表示网络已经中断。
   -- 控制面板》网络和Internet》网络和共享中心
3. 重新打开UltraEdit软件
4. 点击输入许可证密钥，需要创建一个ID和密码，然后点击激活，连接错误选择脱机激活，输入创建的ID及密码，窗口先不要关闭放在一边
5. 打开 UltraEdit 注册机(IDM+All+Products+KeyGen+v3.6.zip)。把生成的用户代码相应的复制到注册机的 User Code 里，点击下侧的 generate, 生成的代码复制到激活程序的验证码框里. 简单来说就是把左边的复制到右边，生成新的代码，然后复制到左边。
6. 弹出激活成功的窗口。表示UltraEdit破解成功！可以打开网络！
7. 重新启用网络: 控制面板》网络和Internet》网络和共享中心》点击左侧“更改适配器设置”, 可以看到已经被禁用的网络，点击可重新启用
