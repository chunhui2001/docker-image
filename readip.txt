

172.16.0.0/16 和172.16.0.0/12的具体区别是什么？ 其实就是/16和/12的区别： 

》〉》〉》〉/12的意思是子网遮罩是12位. 也就是子网遮罩是: 255.240.0.0
》〉》〉》〉255.255.255.255 表示成2进制是: 11111111.11111111.11111111.11111111, 这里的/12告诉你从左边取12个1, 其它的就全用0表示, 那么就是: 11111111.11110000.00000000.00000000. 再把这个换算成10进制子网遮罩就是: 255.240.0.0.

/16也是同理，从左边取16个1.

# On a bridge network you can only create a single subnet:
$ docker network create \
  --driver=bridge \
  --subnet=172.16.197.0/24 \
  br0

$ docker network create \
  --driver=bridge \
  --subnet=172.28.0.0/16 \
  --ip-range=172.28.5.0/24 \
  --gateway=172.28.5.254 \
  br0

$ docker network create -d overlay \
  --subnet=192.168.1.0/25 \
  --subnet=192.170.2.0/25 \
  --gateway=192.168.1.100 \
  --gateway=192.170.2.100 \
  --aux-address="my-router=192.168.1.5" --aux-address="my-switch=192.168.1.6" \
  --aux-address="my-printer=192.170.1.5" --aux-address="my-nas=192.170.1.6" \
  my-multihost-network
